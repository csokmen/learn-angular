12. Pipes

Pipes Example

Built-in Pipes

  • DatePipe: {{ today | date:'fullDate' }}
  • UpperCasePipe: {{ 'hello world' | uppercase }}
  • LowerCasePipe: {{ 'HELLO WORLD' | lowercase }}
  • CurrencyPipe: {{ price | currency:'USD':'symbol' }}
  • PercentPipe: {{ 0.75 | percent }}
  • JsonPipe: 
    {{ someObject | json }}

Custom Truncate Pipe

Original: {{ longText }}

Truncated (default 20 chars): {{ longText | truncate }}

Truncated (50 chars): {{ longText | truncate:50 }}
